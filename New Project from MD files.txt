
  # Create new project
  ng new my-new-app --standalone --routing --style=css
  cd my-new-app

  # Copy only the docs
  cp /path/to/workflow-app/README.md .
  cp -r /path/to/workflow-app/docs .

  Then use those files as your blueprint to build from scratch — either manually or by feeding each doc section to Claude as a prompt.   

  One thing worth noting: the docs I generated describe the current app. If your new project has different requirements, update the docs 
  first before building — treat them as a living spec, not a frozen snapshot. That's the real value: you can edit docs/models.md to add a
   new interface, or docs/api-reference.md to add an endpoint, and then build to that updated spec.
